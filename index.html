Pseudo Elements:

A CSS pseudo-element is a keyword added to a selector that lets you style a specific part of the selected element(s).
For example, it can be used to:

  • Style the first letter, or line, of an element
  • Insert content before, or after, the content of an element

Syntax

                    
                        selector::pseudo-element {
                            property: value;
                        }

Examples

  • first-line: Selects the first line.
  • first-letter: Selects the first letter.
  • before: Insert something before the content.
  • after: Insert something after the content.
  • marker: Selects the markers of list items.
  • selection: Selects the portion of an element that is selected by a user.

Pseudo Classes:

A pseudo-class is used to define a special state of an element.
For example, it can be used to:

  • Style an element when a user mouses over it
  • Style visited and unvisited links differently
  • Style an element when it gets focus

Syntax:

                    
                        selector:pseudo-class {
                            property: value;
                        }

Examples

  • Anchor Pseudo-classes: :link, :visited, :hover, :active
  • :first-child: pseudo-class matches a specified element that is the first child of another element.
  • :last-child: Selects element that is the last child of its parent.
  • nth-child(n): Selects the nth element of its parent.

:is Pesudo Class

                    
                        div:is(a, p, .card, #id) {
                            ...
                        }

                        :is(header, .card) p:hover {
                            ...
                        }

Flexbox

A CSS3 layout mode that provides an easy and clean way to arrange items within a container.

  • No floats!
  • Responsive and mobile friendly
  • Positioning child elements is much easier
  • Flex container's margins do not collapse with the margins of its contents
  • Order of elements can be easily changed without editing the source HTML

Flexbox Model Concenpt

  • The ability to alter item width and height to best fit in its containers available free space
  • Flexbox is direction-agnostic, means it works for both horizetal axis and vertical axis
  • Built for small scale layouts while the upcoming "Grid" specification is for more large scale

Flex Properties

  • display: flex | inline
  • flex-direction: defines the direction of the child elements, its a parent property. row(default) | column
  • flex-wrap: if the screen sizes changes and you want the content to be wraped, use the "wrap" property. wrap | nowrap | wrapreverse
  • flex-basis: defines the width of the child element like "width" property. But it is more flexy.
    Rather than "width" use "flex-basis" in "flexboxes" <length>
  • justify-content: Aligns the child elements horizantally. By defalaul, the child elements takes the whole width of the flexbox, first we will
    give width to the child elements so that the parent container can have some empty space. Now we can
    arrange the child elements horizentally by this property. flex-start(default) | flex-end | center |
    space-between (gives margins between the child elements) | space-around (give margins on both sides of child element)
  • align-self: felx-start | flex-end | center
  • align-items: Aligns the child elements vertically. By default, the height of all child elements is same (stretch), if you want to give it the height
    according to the content in it, use this parent property. flex-start | flex-end | center | stretch
  • align-content: flex-start | flex-end | center
  • flex-grow: <length>
  • flex-shrink: <length>
  • flex: defines length of the child item. <length>
  • order: defines order of the child item, will everyone its oreder to work perfectly. <length>

Grid

Grid allows us to create two dimentional layouts on a webpage, and align items in
both columns and rows.

  • display: gird - Initiates the grid property
  • grid-template-columns: Parent property, give widths to each child to see the effect
  • grid-column-gap (now column-gap): defines the gape between columns. values: any css unit like 1em
  • grid-row-gap: defines the gape between rows. values: any css unit like 1em
  • grid-gap: shorthand for grid-column-gap and grid-row-gap
  • grid-auto-rows: defines the fixed height of the elements in the rows. For flexible height
    use grid-auto-rows: minmax(100px, auto); - By default the height will be 100px but if the content
    goes beyond that height, it will stretch out to fill the content.
  • jsutify-items: Align items horizantally. start | center | end | stretch (default)
  • align-items: Align items vertically. start | center | end | stretch (default)
  • align-self and justify-self: Child property which aligns the child elements both horizantally and vertically.
  • grid-column: Stretches the child element horizantally. e.g: grid-column: 1/3; stretches the element from one to three horizantally.
  • grid-row: Stretches the child element vertically. e.g: grid-row: 1/3; stretches the element from one to three vertically.

In "grid-template-columns" use "fractions" instead of other CSS units as other units you gonna run into margins issues.
1fr 2fr means the second column will be double of the first column.
Examples:
- 1fr 2fr 1fr - First and third column will be same width and the second will be the double
- 1fr 1fr 1fr = repeat(3, 1fr) - Repeat three times 1fr.
- 1fr 2fr 1fr 2fr = repeat(4, 1fr 2fr)

Media Queries

- Allows us to create CSS rules based on various parameters (viewport size, device type etc).
- We can arrange our layout to suite a wide variety of different / screen sizes.

Syntax:

@media <media-type> and (expression) { CSS code }
- media-types: scree | print | TV | speech or all

Example:

- @media TV and (min-width: 1200px) {}
- @media screen and (max-width: 800px) {} - apply the CSS when the screen is 800px and less, if the screen is bigger than 800px then do not apply the rules.
- @media (orientation: landscape/portrait) and (max-width: 700px) {}
- @media (orientation: portrait) {}
